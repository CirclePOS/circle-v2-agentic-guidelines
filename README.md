```

runtime/skills/    ← Skill definitions (/tdd-workflow, /rspec-runner, etc.)
standards/         ← Code and testing standards
workflow/          ← Issue → PR → deploy workflows
architecture/      ← Patterns and architectural decisions
docs/              ← Setup guides and reference docs
```
